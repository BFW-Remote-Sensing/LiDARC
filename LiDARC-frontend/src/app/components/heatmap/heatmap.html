Heatmap Navigation Guide

Use the controls above to switch views and explore areas in detail.

1) Visible Heatmaps
The toggle group selects which heatmaps are displayed:

  • A + B + Δ: show both datasets and the difference heatmap.
  • A + B: show the two linked datasets side-by-side.
  • Δ: show only the difference heatmap (B − A).

2) Legend (VisualMap)
The VisualMap button shows/hides the color legend (value → color mapping).

3) Zoom Controls
The Zoom button shows/hides the zoom sliders:

  • Horizontal Zoom: use the bottom slider to zoom along the X-axis.
  • Vertical Zoom: use the right slider to zoom along the Y-axis.

4) Color Scheme
The Scheme button allows you to select different color schemes for the heatmaps.
