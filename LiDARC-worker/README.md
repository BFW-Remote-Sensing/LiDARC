# LiDARC Worker

# Get started
1. (Recommended) Set up a venv, uv, or conda-environment with Python >= 3.11
2. Install all requirements `pip install -r src/preprocess/requirements.txt` or `pip install -r src/metadata/requirements.txt`
3. Run `python -m pytest`. All test should be successful.

## VENV
To create a venv just use `python -m venv .venv` which creates a directory called ``.venv``
To activate use `.venv/Scripts/Activate` or `.venv/Scripts/activate.ps1` for powershell
